Комментарии:

- Являются важной частью кода, потому что позволяют описать определенный участок кода.
- Комментарии записываются с помощью символа #.


Функция print:

- Является функцией, которая позволяет выводить текст на экран.


Форматы вывода:

- Дополнительное свойство sep является разделителем между значениями при выводе.
- Дополнительное свойство end является устанавливателем конца строки.


Специальные символы:

\n - перевод на новую строку.
\  - делает необычные символы обычными.
\t - делает табуляцию.


Математические действия:

+ - сложение.
- - вычитание.
* - умножение.
/ - деление.
% - остаток от деления.
** - возведение в степень.
// - целочисленное деление.


Математические функции:

- min позволяет найти минимальный элемент среди данных элементов.
- max позволяет найти максимальный элемент среди данных элементов.
- abs позволяет найти число по модулю.
- pow позволяет возвести некое число в определенную степень.
- round позволяет округлять числа.


Команда input:

- Является командой для получения данных от пользователя.


Переменные:

- Являются ячейкой памяти, в которую можно занести некую информацию и далее к этой информации ссылаться.
- Создаются с помощью имени и значения.
- Типы данных:
              int - тип данных, который отвечает за целые числа.
              float - тип данных, который отвечает за числа с плавающей точкой.
              string - тип данных, который отвечает за строки.
              bool - тип данных, который отвечает за значения True или False.
