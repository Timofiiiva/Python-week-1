
1. Значение sep по ум. 
- Значение sep по умолчанию является дополнительным свойством, которое разделяет элементы.

2. Значение end по ум. 
- Значение end по умолчанию является дополнительным свойством, которое устанавливает конец для строки.

3. Что делает специальный символ \ ?
- Специальный символ \ делает не обычные символы обычными например: print("\"").

4. Что делает специальный символ \n ?
- Специальный символ \ n делает перевод на новую строку.

5. Что делает специальный символ \t ?
- Специальный символ \t  делает табуляцию.

6. Что делает математический оператор % ?
- Математический оператор % делает получение остатка при деление. 

7. Что делает математический оператор // ?
- Математический оператор // делает возведение в степень.

8. Что делает математический оператор ** ?
- Математический оператор // делает деление в целое число.

9. Что возвращает abs(-2) ?
- Функция abs(-2) возвращает результат положительное значение числел: (2).

10. Что возвращает pow(2, 4) ?
- Функция pow(2, 4) возвращает результат возведения некоторого числа в определенную степень: (16).

11. Что возвратит round(5 / 2) ?
- Функция round(5 / 2) возвращает результат деления и округления чисел: (3).

12. Зачем нужна директива del ?
- Директива del нужна для того, чтобы удалять переменные.

13. Создайте переменную "age" и присвойте ей значение вашего возраста. Выведите ее значение.
Затем используйте директиву "del" для удаления этой переменной, и попробуйте вывести значение снова.
Что произошло?
- Возникла ошибка NameError: name 'age' is not defined, 
потому что сначала переменная была создана, а затем была удалена.
Далее была попытка обратиться к переменной, которой уже не существует.
